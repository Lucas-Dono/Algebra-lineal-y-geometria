# Forma Polar y Exponencial

La representación polar de los números complejos revela la estructura geométrica de la multiplicación y división, convirtiéndolas en operaciones simples sobre ángulos y longitudes.

## Coordenadas Polares

Todo punto en el plano (excepto el origen) se puede describir mediante:
- Su **distancia** al origen: $r \geq 0$
- Su **ángulo** respecto al eje real positivo: $\theta$

Para un número complejo $z = a + bi$:

$$r = |z| = \sqrt{a^2 + b^2}$$

$$\theta = \arg(z) = \arctan\left(\frac{b}{a}\right) \quad \text{(ajustando por cuadrante)}$$

## Forma Polar

La **forma polar** de un número complejo es:

$$z = r(\cos\theta + i\sin\theta)$$

donde:
- $r = |z|$ es el **módulo**
- $\theta = \arg(z)$ es el **argumento**

### Conversión de Cartesiana a Polar

Dado $z = a + bi$:

1. Calcular $r = \sqrt{a^2 + b^2}$

2. Calcular $\theta$ según el cuadrante:

| Cuadrante | Condición | Fórmula |
|-----------|-----------|---------|
| I | $a > 0$, $b \geq 0$ | $\theta = \arctan(b/a)$ |
| II | $a < 0$, $b \geq 0$ | $\theta = \pi + \arctan(b/a)$ |
| III | $a < 0$, $b < 0$ | $\theta = -\pi + \arctan(b/a)$ |
| IV | $a > 0$, $b < 0$ | $\theta = \arctan(b/a)$ |
| Eje $+$ | $a > 0$, $b = 0$ | $\theta = 0$ |
| Eje $-$ | $a < 0$, $b = 0$ | $\theta = \pi$ |

**Ejemplo:** Convertir $z = 1 + \sqrt{3}i$ a forma polar.

$$r = \sqrt{1^2 + (\sqrt{3})^2} = \sqrt{1 + 3} = 2$$

Como $a = 1 > 0$ y $b = \sqrt{3} > 0$ (primer cuadrante):
$$\theta = \arctan\left(\frac{\sqrt{3}}{1}\right) = \arctan(\sqrt{3}) = \frac{\pi}{3}$$

Forma polar: $z = 2\left(\cos\frac{\pi}{3} + i\sin\frac{\pi}{3}\right)$

### Conversión de Polar a Cartesiana

Dado $z = r(\cos\theta + i\sin\theta)$:

$$a = r\cos\theta$$
$$b = r\sin\theta$$

Por lo tanto: $z = r\cos\theta + ir\sin\theta$

## Forma Exponencial

La **fórmula de Euler** establece:

$$e^{i\theta} = \cos\theta + i\sin\theta$$

Esta identidad conecta las funciones exponencial y trigonométricas.

La **forma exponencial** de un número complejo es:

$$z = re^{i\theta}$$

**Ejemplo:** $z = 2e^{i\pi/3}$ representa el mismo número que $2(\cos\frac{\pi}{3} + i\sin\frac{\pi}{3})$.

### Identidad de Euler

Un caso especial hermoso:

$$e^{i\pi} + 1 = 0$$

Esta ecuación relaciona cinco constantes fundamentales: $e$, $i$, $\pi$, $1$ y $0$.

## Multiplicación en Forma Polar

La multiplicación es donde la forma polar brilla:

$$z_1 \cdot z_2 = r_1 e^{i\theta_1} \cdot r_2 e^{i\theta_2} = r_1 r_2 e^{i(\theta_1 + \theta_2)}$$

En otras palabras:

$$|z_1 z_2| = |z_1| \cdot |z_2|$$
$$\arg(z_1 z_2) = \arg(z_1) + \arg(z_2)$$

**Interpretación geométrica:** Multiplicar por $z$ significa:
- Escalar por $|z|$
- Rotar por $\arg(z)$

**Ejemplo:** Calcular $(1 + i)(1 + \sqrt{3}i)$

En forma polar:
- $1 + i = \sqrt{2}e^{i\pi/4}$
- $1 + \sqrt{3}i = 2e^{i\pi/3}$

Producto:
$$(\sqrt{2}e^{i\pi/4})(2e^{i\pi/3}) = 2\sqrt{2}e^{i(\pi/4 + \pi/3)} = 2\sqrt{2}e^{i \cdot 7\pi/12}$$

## División en Forma Polar

$$\frac{z_1}{z_2} = \frac{r_1 e^{i\theta_1}}{r_2 e^{i\theta_2}} = \frac{r_1}{r_2} e^{i(\theta_1 - \theta_2)}$$

Es decir:

$$\left|\frac{z_1}{z_2}\right| = \frac{|z_1|}{|z_2|}$$
$$\arg\left(\frac{z_1}{z_2}\right) = \arg(z_1) - \arg(z_2)$$

## Fórmula de De Moivre

Para calcular potencias de números complejos:

$$z^n = [r(\cos\theta + i\sin\theta)]^n = r^n(\cos(n\theta) + i\sin(n\theta))$$

O en forma exponencial:

$$(re^{i\theta})^n = r^n e^{in\theta}$$

**Ejemplo:** Calcular $(1 + i)^8$

Primero, convertir a forma polar:
- $|1 + i| = \sqrt{2}$
- $\arg(1 + i) = \pi/4$

Entonces $1 + i = \sqrt{2}e^{i\pi/4}$

Aplicando De Moivre:
$$(1 + i)^8 = (\sqrt{2})^8 e^{i \cdot 8 \cdot \pi/4} = 16 e^{i \cdot 2\pi} = 16 \cdot 1 = 16$$

Verificación: $(1+i)^2 = 2i$, $(2i)^2 = -4$, $(-4)^2 = 16$.

## Argumento Principal

El argumento de un número complejo no es único: si $\theta$ es un argumento, también lo es $\theta + 2\pi k$ para cualquier entero $k$.

El **argumento principal** $\text{Arg}(z)$ es el único argumento en el intervalo $(-\pi, \pi]$.

**Ejemplo:** Para $z = -1$:
- Argumentos válidos: $\pi$, $3\pi$, $-\pi$, $5\pi$, ...
- Argumento principal: $\text{Arg}(-1) = \pi$

## Propiedades del Argumento

1. $\arg(z_1 z_2) = \arg(z_1) + \arg(z_2)$ (módulo $2\pi$)
2. $\arg(z_1/z_2) = \arg(z_1) - \arg(z_2)$ (módulo $2\pi$)
3. $\arg(z^n) = n \cdot \arg(z)$ (módulo $2\pi$)
4. $\arg(\bar{z}) = -\arg(z)$
5. $\arg(z) = 0 \iff z$ es real positivo
6. $\arg(z) = \pi \iff z$ es real negativo

## Casos Especiales

| Número | Módulo | Argumento |
|--------|--------|-----------|
| $1$ | $1$ | $0$ |
| $-1$ | $1$ | $\pi$ |
| $i$ | $1$ | $\pi/2$ |
| $-i$ | $1$ | $-\pi/2$ |
| $1 + i$ | $\sqrt{2}$ | $\pi/4$ |
| $1 - i$ | $\sqrt{2}$ | $-\pi/4$ |

## Aplicaciones

### Rotaciones en el Plano

Multiplicar por $e^{i\theta}$ rota un punto $\theta$ radianes en sentido antihorario.

Para rotar el punto $z$ un ángulo $\alpha$:
$$z' = z \cdot e^{i\alpha}$$

### Vibraciones y Ondas

Las señales sinusoidales se representan como:
$$A\cos(\omega t + \phi) = \text{Re}(Ae^{i(\omega t + \phi)})$$

La forma exponencial simplifica el análisis de circuitos eléctricos y sistemas vibratorios.
